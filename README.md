# Translator #ID 2410237 Prince Zulu
To setup, first download all the files then put them in one folder you can name Translator.
Then click on the file named index and open it in your browser or it will open automatically if its by default already.
After it opens, choose a language you want to translate and enter the text in the text box and it will be translation for you while you type. If not then click the translate button.
Make sure you are connected to the internet since API's are being used to make translations possible
